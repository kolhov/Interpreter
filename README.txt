To get started, you need to have Node.js installed.
https://nodejs.org/en

You can check the installation by running the following command:
    node -v
This command will display the installed Node.js version if it's correctly set up.

You need to install all dependencies by running the following command:
    npm install

To run the script, run the following command:
    npm start


The file to be processed should be located in the input folder,
and the result will be saved in the output folder.
